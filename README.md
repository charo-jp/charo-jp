「スキューバダイビング」の魅力

「スキューバダイビング」を最近始めてみました。
実は私、去年の12月に友人と一緒に、
ダイビングの資格を取得したばかりですが、ダイビングの資格とダイビングの魅力を3つに分けてお伝えします！

ダイビングの資格とは?
「ダイビングって免許がいるの？」とよく聞かれるのですが、実は車の免許のような国家資格ではなく、
民間のスクールなどで講習を受けるともらえる資格（認定証）なんです。
これがあると「世界の海で安全に潜れる知識と技術がありますよ」という証明になります。

面白いのは、ランクによって「潜れる深さ」が変わることです！

最初のランク（私が取ったもの）
水深18メートル（ビルで言うと5〜6階くらい）まで潜れます。これだけでも、
浅瀬の綺麗なサンゴ礁やたくさんの魚たちを十分楽しめます！

次のステップのランク
さらにステップアップすると、水深30メートル（ビル10階分相当！）という深海の世界まで潜れるようになります！
ここまで行くと、沈没船やを見に行ったり、ちょっと大きな海の生き物に出会えたりと、冒険の幅がグッと広がります！
実は今月、この「水深30メートルまで行ける資格」も取得しに行く予定です！

実際に潜ってわかった！ダイビング3つの魅力

1. 水中で「息ができる」
ダイビングを始めて誰もが最初に感動するのが、「水中で普通に呼吸ができる」ということです！
最初は少しドキドキして本当に息できるのかと思っていましたが、海の中で息を吸えた瞬間は、
これまでにない新鮮な感覚で本当に感動しました！

2. 冬の海でも寒くない！？「ドライスーツ」の魔法
12月にダイビングをしましたが、そのまま海に入ると寒すぎるので冬の間は「ドライスーツ」を着用します
このスーツ、なんと海の中に入っても体が全く濡れません。
スーツの中に空気を送り込んで空気の層を作るので、真冬の海でも冷たさを感じることなく、
快適にダイビングをすることが出来ます。

3. 自然のタコや魚を間近に見れる
ライセンスを取得した場所は、「伊豆」でダイビングスポットとしてとても有名です。
冬の伊豆の海は驚くほど透明度が高く、本当に綺麗で、海の中にはたくさんの魚たちが泳いでいていました！


最後に
ダイビングは、日常の忙しさを忘れて、ただただ目の前にある自然の綺麗さに癒やされる最高のリフレッシュ方法だなと感じています。

伊豆は都心からのアクセスも良く、初心者でも楽しめる素晴らしいスポットがたくさんあります。
「何か新しい趣味を始めたいな」「綺麗な海に癒やされたいな」と思っている方がいれば、ぜひ挑戦してみてください！
